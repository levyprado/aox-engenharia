# AOX Engenharia

A modern, high-performance landing page for **AOX Engenharia**, a civil engineering firm. This project showcases the company's services, portfolio, and commitment to technical responsibility.

![Next.js](https://img.shields.io/badge/Next.js-000?style=for-the-badge&logo=nextdotjs&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)

## 🚀 Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Service Catalog:** Detailed sections for Residential, Commercial, and Solar energy projects.
- **Project Gallery:** Visual showcase of completed works with hover effects.
- **Workflow Timeline:** A step-by-step "How We Work" section to build client trust.
- **Direct Integration:** One-click contact via WhatsApp, Email, and Phone.
- **Performance:** Built with Next.js App Router for SEO-friendly, lightning-fast loading.

## 🛠️ Tech Stack

- **Framework:** [Next.js 16+](https://nextjs.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons:** [Phosphor Icons](https://phosphoricons.com/)
- **Fonts:** Manrope (via `next/font`)
- **Deployment:** Vercel
